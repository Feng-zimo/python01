# 布尔值运算

- True 相当于 1，False 相当于 0
- True 必须大写，`true` 会被当做变量名

```python
print(True + 1)   # 输出：2
print(False + 1)  # 输出：1
print(True + False) # 输出：1
```

# 变量命名规则

- 变量名由字母、数字、下划线组成

# if 判断

```python
# 基本结构
if 条件:
    执行语句
elif 条件:
    执行语句
else:
    执行语句
```

注意：条件判断从上往下执行，一旦满足某个条件就不会再往下判断。精确匹配（如 `==60`）要放在范围匹配（如 `>=60`）前面。

# 字符串切片

语法：`[开始位置:结束位置:步长]`，包前不包后。

- 步长默认 1，绝对值决定间隔，正负号决定方向（正数从左往右，负数从右往左）

```python
st = "g79j468abc789"
print(st[0:4])   # g79j
print(st[7:])    # abc789
print(st[:7])    # g79j468
print(st[-1:])   # 9
```

# 列表

## 定义

```python
li = [1, 2, 3, 4]           # 基本定义
li = [1, 2, "hello", True]  # 可包含任意类型
```

## 常见操作

| 操作             | 描述                       | 示例                              |
| ---------------- | -------------------------- | --------------------------------- |
| `append(x)`      | 末尾添加一个元素           | `li.append('four')`               |
| `extend(list)`   | 末尾添加多个元素（传列表） | `li.extend(['five','six'])`       |
| `insert(i, x)`   | 指定位置添加元素           | `li.insert(0, 'zero')`            |
| `li[i] = x`      | 通过下标修改元素           | `li[0] = 'zero'`                  |
| `del li[i]`      | 删除指定下标元素           | `del li[0]`                       |
| `pop()`          | 删除末尾元素               | `li.pop()`                        |
| `pop(i)`         | 删除指定下标元素           | `li.pop(0)`                       |
| `remove(x)`      | 删除第一个匹配的元素       | `li.remove(2)`                    |
| `clear()`        | 清空列表                   | `li.clear()`                      |
| `sort()`         | 升序排序                   | `li.sort()`                       |
| `reverse()`      | 反转列表                   | `li.reverse()`                    |
| `in` / `not in`  | 判断元素是否存在           | `'one' in li`                     |
| `index(x)`       | 查找元素索引               | `li.index('one')`                 |
| `count(x)`       | 统计元素出现次数           | `li.count(1)`                     |

## 列表推导式

```python
# 基本语法：[表达式 for 变量 in 可迭代对象]
li = [i for i in range(1, 10)]

# 带条件：[表达式 for 变量 in 可迭代对象 if 条件]
li = [i for i in range(1, 10) if i % 2 == 1]
```

## 列表嵌套

```python
li = [1, 2, [3, 4, [5, 6]]]
print(li[2][2][0])  # -> 5
```

# 元组

- 元组是不可变类型，不能修改元素
- 只有一个元素时必须加逗号：`t = (1,)`，否则 `(1)` 是整数

```python
t = (1, 2, 3, 4)
print(t[0])  # -> 1
# 支持 count(), index(), len(), in, not in
```

应用场景：函数参数和返回值、格式化输出的 `()` 本质是元组、保护数据安全。

# 字典

## 定义

```python
dic = {"name": "bingbing", "age": 18}
# 键具备唯一性（重复则后面覆盖前面），值可以重复
```

## 常见操作

| 操作              | 描述                       | 示例                          |
| ----------------- | -------------------------- | ----------------------------- |
| `dic[key]`        | 获取值（键不存在则报错）   | `dic['name']`                 |
| `dic.get(key)`    | 获取值（键不存在返回None） | `dic.get('name')`             |
| `dic.get(key,默认值)` | 获取值（可设默认值）   | `dic.get('gender', "未知")`   |
| `dic[key] = val`  | 添加/修改键值对            | `dic['name'] = "susu"`        |
| `del dic[key]`    | 删除键值对                 | `del dic['name']`             |
| `dic.pop(key)`    | 删除指定键值对             | `dic.pop('name')`             |
| `dic.popitem()`   | 删除最后一个键值对         | `dic.popitem()`               |
| `dic.clear()`     | 清空字典                   | `dic.clear()`                 |
| `len(dic)`        | 键值对数量                 | `len(dic)`                    |
| `dic.keys()`      | 返回所有键                 | `dic.keys()`                  |
| `dic.values()`    | 返回所有值                 | `dic.values()`                |
| `dic.items()`     | 返回所有键值对             | `dic.items()`                 |

```python
for key in dic.keys():     # 遍历键
for val in dic.values():   # 遍历值
for item in dic.items():   # 遍历键值对
```

# 集合

- 元素唯一，不能重复；无序（基于哈希表实现）
- 可用于列表/元组去重

```python
set1 = {1, 2, 3, 4, 5}
set_empty = set()   # 空集合必须用 set()，{} 是空字典
```

## 常见操作

| 操作             | 描述                   |
| ---------------- | ---------------------- |
| `add(x)`         | 添加元素               |
| `update(list)`   | 添加多个元素           |
| `remove(x)`      | 删除元素（不存在报错） |
| `discard(x)`     | 删除元素（不存在不报错）|
| `pop()`          | 随机删除元素           |
| `clear()`        | 清空集合               |

## 集合运算

| 运算            | 描述     | 示例          |
| --------------- | -------- | ------------- |
| `set1 & set2`   | 交集     | 两个集合共有  |
| `set1 | set2`   | 并集     | 所有元素去重  |
| `set1 - set2`   | 差集     | set1 独有     |

# 类型转换

| 函数       | 描述                             | 示例                    | 注意事项                           |
| ---------- | -------------------------------- | ----------------------- | ---------------------------------- |
| `int()`    | 转整数（舍弃小数部分）           | `int(1.2)` → 1          | 字符串必须是纯数字（允许前导+-号） |
| `float()`  | 转浮点数（自动补 .0）            | `float(1)` → 1.0        | 字符串必须是纯数字                 |
| `str()`    | 转字符串（任何类型都可以）       | `str(100)` → "100"      | 浮点转字符串会去掉末尾多余的0      |
| `eval()`   | 将字符串当代码执行               | `eval("10+10")` → 20    | 强大但不安全，不建议使用           |
| `list()`   | 可迭代对象转列表                 | `list("123")` → ['1','2','3'] | int/float 不可转              |
| `tuple()`  | 可迭代对象转元组                 | `tuple("123")` → ('1','2','3') | int/float 不可转              |

> `eval()` 可实现 list、dict、tuple 与 str 之间的互相转换。

# 赋值与拷贝

| 方式              | 描述                                     | 效果                           |
| ----------------- | ---------------------------------------- | ------------------------------ |

| `li2 = li.copy()` | 浅拷贝，创建新的内存地址                 | 修改 li，li2 不受影响          |
